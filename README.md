# todoApp-team

## ローカル環境構築手順

1. /backend ディレクトリへ移動
   `cd /backend`

2. 以下コマンド実行
   `npm install`

3. /frontend ディレクトリへ移動
   `cd /frontend`

4. 以下コマンド実行
   `npm install`

5. backend サーバー起動
   `npm start`

6. frontend サーバー起動
   `npm start`

7. /db ディレクトリへ移動
   `cd /db`

8. db サーバー起動
   `docker-compose up -d`
